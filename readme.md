# Meteo Blog

A meteo blog API, written on FastAPI + SQLAlchemy + Pydantic.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Demo](#demo)

## Installation

Download to your project directory, add run:

```sh
pip install -r requirements.txt
```

## Usage

Run
```sh
uvicorn main:app
```
then go to http://127.0.0.1:8000/docs
